# 设计规范

> 最后更新：2026-07-22
> 版本：v1.0

---

## 设计理念

**关键词**：温暖、活泼、食物、治愈

设计目标是让用户感到「这个助手很可爱」，降低选择困难带来的焦虑感。

## 配色方案

### 主色调

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色（暖橙） | `#FF6B35` | 按钮、标题、进度条、气泡（AI） |
| 辅色（奶黄） | `#FFF3E0` | 页面背景 |
| 强调色（薄荷绿） | `#26A69A` | 用户气泡、完成状态 |
| 文字色 | `#333333` | 正文 |
| 弱文字 | `#999999` | 提示文字 |

### 功能色
| 角色 | 色值 | 用途 |
|------|------|------|
| 成功 | `#66BB6A` | 推荐完成 |
| 错误 | `#EF5350` | 错误提示 |
| 加载 | `#FFB74D` | 等待动画 |

## 字体

```css
font-family: "PingFang SC", "Noto Sans SC", "Microsoft YaHei", system-ui, sans-serif;
```

- 标题：`bold 24px`
- 正文：`regular 16px`
- 辅助文字：`regular 14px`
- 小字提示：`regular 12px`

## 布局

### 手机端（默认）
```
┌─────────────────────┐
│   🍔 AI 外卖推荐     │  ← 固定顶栏
├─────────────────────┤
│                     │
│  [AI 气泡]          │
│  今天想吃什么口味？  │
│                     │
│         [用户气泡]   │
│         想吃辣的～   │
│                     │
│  [AI 气泡]          │
│  ...                │
│                     │
│  进度: ●●●○○ 5/10    │
├─────────────────────┤
│ [输入框]      [发送] │  ← 固定底栏
└─────────────────────┘
```

### 桌面端
- 最大宽度 600px 居中
- 保持与手机端相同的布局逻辑

## 动画规范

| 元素 | 动画 | 时长 |
|------|------|------|
| 页面加载 | 食物 emoji 从底部飘入 | 800ms |
| AI 气泡出现 | 滑入 + 淡入 | 300ms |
| AI 文字 | 打字机逐字显示 | 30ms/字 |
| 用户气泡出现 | 从右侧滑入 | 200ms |
| 进度条 | 填充动画 | 400ms |
| 按钮悬停 | 轻微放大 1.05x | 150ms |
| 加载状态 | 三个点跳动 | 循环 1s |
| 推荐结果 | 卡片翻转出现 | 500ms |

## 组件规范

### 聊天气泡
- AI 气泡：左对齐，圆角 `16px 16px 16px 4px`，暖橙背景 `#FF6B35`，白色文字
- 用户气泡：右对齐，圆角 `16px 16px 4px 16px`，浅灰背景 `#F5F5F5`，深色文字
- 最大宽度 80%，间距 12px

### 进度条
- 高度 6px，圆角 3px
- 底色 `#E0E0E0`，填充色 `#FF6B35`
- 右侧显示数字 `5/10`

### 输入区
- 输入框：圆角 `24px`，白色背景，边框 `#E0E0E0`
- 发送按钮：圆形，暖橙色，内含箭头 icon
- 聚焦态：边框变为暖橙色

### 推荐卡片
- 白色卡片，圆角 `20px`，阴影 `0 4px 20px rgba(0,0,0,0.1)`
- 顶部大 emoji 🍕
- 菜系标签 + 菜品名称 + 理由 + 趣味评语
- 底部「再来一次」按钮

## 对话语气规范

- 使用「你」而非「您」（亲切）
- 每句话配 0-2 个 emoji
- 可用的食物 emoji 库：🍕🍔🌮🍣🍜🍛🥘🍱🧋🍰
- 每轮问题控制在 30 字以内
- 推荐结果控制在 100 字以内

## 状态 UI

| 状态 | 视觉表现 |
|------|----------|
| 初始欢迎 | 大 logo + 食物飘落动画 + 「开始推荐」按钮 |
| 等待回复 | 三个点跳动动画 + 「饭饭正在思考...」 |
| 对话进行 | 聊天气泡逐条显示 |
| 推荐完成 | 推荐卡片 + 撒花动画 |
| 错误 | 红色提示条 + 重试按钮 |
