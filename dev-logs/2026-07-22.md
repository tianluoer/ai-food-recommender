### Vercel API 函数无法运行
- **现象**：所有 API 端点返回 `FUNCTION_INVOCATION_FAILED`
- **已验证**：环境变量 `DEEPSEEK_API_KEY` 已设置（Production + Preview）
- **已验证**：代码推送到 GitHub，部署 10s 内完成，状态 Ready
- **已验证**：极简测试端点（无外部依赖）同样失败
- **排查方向**：
  1. Vercel 框架预设为 "Other"，但可能缺少 `@vercel/node` 构建器声明
  2. 项目缺少 `build` 脚本和 `public` 目录 → Vercel 无法正确识别静态+函数混合项目
  3. 可能需要 In `vercel.json` 明确声明构建器：
     ```json
     {
       "builds": [
         { "src": "api/**/*.js", "use": "@vercel/node" },
         { "src": "*.html", "use": "@vercel/static" }
       ]
     }
     ```
